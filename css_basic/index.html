meta charset="utf-8" The meta element. This element represents metadata that cannot be represented by other HTML meta-related elements, like base, link, script, style or title. The charset attributes sets the character set for your document to UTF-8, which includes most characters from the vast majority of human written languages. With this setting, the page can now handle any textual content it might contain. There is no reason not to set this, and it can help avoid some problems later. title The title element. This sets the title of the page, which is the title that appears in the browser tab the page is loaded in. The page title is also used to describe the page when it is bookmarked. body: The body element. This contains all the content that displays on the page, including text, images, videos, games, playable audio tracks, or whatever else.
